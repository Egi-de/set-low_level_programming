# 0x01. C - Variables, if, else, while

## Description

This project covers the use of variables, if/else statements, and while loops in C programming. All programs are written following the Betty coding style and compiled with strict flags on Ubuntu 20.04 LTS.

## Requirements

- Allowed editors: vi, vim, emacs
- Compiled on Ubuntu 20.04 LTS using `gcc -Wall -Werror -Wextra -pedantic -std=gnu89`
- All files end with a new line
- No global variables
- Maximum 5 functions per file
- No use of `system`
- Betty style compliant

## Tasks

| File | Description |
|------|-------------|
| `0-positive_or_negative.c` | Prints whether a random number is positive, negative, or zero |
| `1-last_digit.c` | Prints the last digit of a random number with conditions |
| `2-print_alphabet.c` | Prints the lowercase alphabet using putchar only |
| `3-print_alphabets.c` | Prints lowercase then uppercase alphabet |
| `4-print_alphabt.c` | Prints lowercase alphabet except q and e |
| `5-print_numbers.c` | Prints all single digit numbers 0-9 |
| `6-print_numberz.c` | Prints digits 0-9 using putchar, no char variables |
| `7-print_tebahpla.c` | Prints lowercase alphabet in reverse |
| `8-print_base16.c` | Prints all base 16 numbers in lowercase |
| `9-print_comb.c` | Prints all possible combinations of single-digit numbers |
